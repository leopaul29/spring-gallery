# gallery

> In this project, I made a webapp to consume a gallery API (album and photos).

In this webapp we can
- [x] show a list of albums
- [x] show a single albums with multiple photos
- [ ] show a signle photo
- [ ] import file photo
- [ ] manage cache settings
- [ ] test restclientapi
